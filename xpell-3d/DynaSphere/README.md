# DynaSphere Audio Visualizer with Xpell 3D




## Description
This project is a 3D Audio Visualizer using Xpell 3D and Three.js. It features a dynamic 3D sphere that reacts to audio input, changing its size and color in real time. The Web Audio API is used to capture audio, and the Xpell library powers the 3D visualization.


Main features:
- Create empty Xpell project
- Create X3DObject named "DynaSphere"
- Add audio visualizer logic to X3DObject `onFrame`
- Add XUI controls for the audio visualizer
- Connect audio input to the visualizer



## Installation

```bash
npm install
```


## Usage

```bash
npm run dev
```
